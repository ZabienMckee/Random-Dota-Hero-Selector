# Random-Dota-Hero-Selector
A simple python program that generates random heroes for the game Dota 2
You will need Python 3.7 or newer to run the code.
Python interpreter can be found here: https://www.python.org/downloads/release/python-370/
report any bugs or questions to zabien.github@gmail.com. A response may take some time.

HOW TO RUN IT.
Install both the code, and Python. 
Open up Python and open the file
click "Run", then "Run Module"
